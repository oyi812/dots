## nvim

v for nvim

<leader>\ sequence to exit insert mode
WW to write file
TT to open terminal
H,J,K,L to move between windows

§ for NERDTree
± to locate current buffer in tree
t open file in tab
gt,<N>gt to move between tabs
cd on directory to change CWD for opening terminal in tab
